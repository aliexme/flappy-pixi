# Flappy Pixi

Flappy Bird game written in pixi.js

[![Netlify Status](https://api.netlify.com/api/v1/badges/5fad5474-8cb0-4755-9240-78d12b459248/deploy-status)](https://flappy-pixi.netlify.app)

## Demo

[Online demo](https://flappy-pixi.netlify.app)

## Installing dependencies

Run the following command to install dependencies.

```
yarn install
```

## Start

Run the following command to run the game locally. The game will start at [`localhost:1234`](http://localhost:1234).

```
yarn start
```

## Build

Run the following command to build the game in production mode.

```
yarn build
```
